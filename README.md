# Lilay's Portfolio

Welcome! This is my personal website and portfolio, built to showcase my projects, interests, and a bit about me.

## 🌍 About This Site

This site is designed as a creative space to introduce myself, highlight my work, and experiment with web technologies. You’ll find pages for my projects, an about section, and a contact form—all wrapped in a unique, visually rich design.

## 🖥️ Tech Stack

- **Next.js (App Router)**
- **React Three Fiber & Drei** for interactive 3D visuals (see the Tux model on the homepage!)
- **TailwindCSS** for responsive, custom styling
- **TypeScript** throughout the codebase
- **OpenNext + Cloudflare** for modern, serverless deployment

## 🎨 Design

The site embraces a Gruvbox-inspired color palette and includes playful UI touches like animated backgrounds and typewriter effects in the navbar.

## 🚀 Getting Started

To run locally:

```bash
npm install
npm run dev
# then visit http://localhost:3000
```
